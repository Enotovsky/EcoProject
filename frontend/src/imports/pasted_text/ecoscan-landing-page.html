Create a modern one-page landing page for a startup project called “EcoScan”. Project idea: EcoScan is a smart carbon footprint calculator based on shopping receipts. A user uploads or photographs a paper receipt, the system recognizes the products, classifies them into ecological categories, calculates an approximate CO₂ footprint, shows an eco-score, and gives simple suggestions for more eco-friendly alternatives. Main goal of the website: Explain the idea clearly, make the product feel fresh and useful, and lead the user to upload a receipt or leave a request. Visual style: Use a bold editorial illustration style inspired by modern magazine covers and abstract urban illustration. The design should feel artistic, playful, smart, and slightly futuristic. Use large flat color shapes, grainy texture, soft gradients, expressive stylized characters, and decorative geometric elements. Avoid boring corporate eco-design. The site should not look like a typical green environmental website. Color palette: Use a dark green or almost black background as the base. Add bright lime green, electric blue, coral pink, warm orange, and off-white. The design should feel high-contrast, stylish, and memorable. Typography: Use a modern clean sans-serif font for interface text. Use a large expressive display font for headings. Headings should be bold, wide, and editorial. Body text should be simple and readable. Layout: Create a desktop landing page in 1440px width. Use a clean 12-column grid. Make the page feel spacious.
